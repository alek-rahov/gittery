HTML-теги контейнеры, такие как "p" и "div", можно использовать как основу для создания разных геометрических фигур (прямоугольники, круги, рамки вокруг абзаца). А с помощью средств CSS этим основам-контейнерам можно придавать цвета, толщину границ, скругления и т.д

Алгоритм:

в блочном теге создаем класс
в CSS обращаемся к этому классу и придаем ему нужные свойства
