ТЕГИ

*все теги заключаются в стрелки <>
**закрывающий тег ставится c символом "/"

ОСНОВНЫЕ ТЕГИ
тег html - основной (контейнер), задает текст, парный
meta - задает кодировку текста
title - название страницы, парный
body - содержимое страницы, парный
br - переход на следующую строку

ТЕГИ ДЛЯ РАБОТЫ СО СПИСКАМИ
ol - нумерованнный список
ul - маркированный список
li - элемент списка

ТЕГИ ДЛЯ РАБОТЫ С ТАБЛИЦАМИ
table - таблица
caption - название таблицы
tr - строка таблицы
td - столбец таблицы c содержимым

ТЕГИ ДЛЯ РАБОТЫ С ТЕКСТОМ
div - блочный тег
span - строчный тег
p - тег создания абзаца
h1, .. h6 - теги заголовков (по уровню)
hr - горизонтальная линия

ТЕГИ ДЛЯ РАБОТЫ С ФОРМАМИ
form - форма
input - ввод данных (данные могут быть разных типов)br button - кнопка
select - выбор
option - элемент выбора внутри тега select
textarea - текстовое поле

АТРИБУТЫ

title - заголовок (подсказка, которая появляется при наведении на тег с этим атрибутом)
class - класс элемента
id - идентификатор элемента
target -задает окно назначения в браузере (где откроется указанная в ссылке страница)
alt - позволяет добавить текстовое описание содержимого картинки

CSS селекторы

*через "." обращение к классу (class), через "# - к идентификатору (id)
