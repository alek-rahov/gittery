Комбинации клавиш для работы в VScode
  • Ctrl + S сохранить файл
  • Alt + Z подровнять текст по левому краю
  • Ctrl + + увеличить текст всего редактора (интерфейс и код)
  • Alt + ЛКМ поставить курсор в нескольких местах
  • Alt + Shift + PgDown копировать строку и вставить ee ниже
  • Ctrl + . ; Ctrl + / создание комментария
  • Ctrl + пробел выбор типа данных (внутри тега)
